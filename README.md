# OIBSIP Web Development and Design Internship

## Level 1 - Task 1 : Landing Page

### Project Title

TechNova - Modern Landing Page

### Objective

The objective of this project is to create an attractive and responsive landing page using HTML and CSS. This project helps in understanding the fundamentals of web development, including page layout design, styling, responsiveness, and user interface design.

### Technologies Used

* HTML5
* CSS3
* Visual Studio Code
* Git & GitHub

### Features

* Responsive Navigation Bar
* Hero Section with Call-to-Action Button
* Services Section
* About Us Section
* Contact Section
* Footer Section
* Mobile Responsive Design
* Modern UI Design

### Steps Performed

1. Created the project structure.
2. Designed the navigation bar.
3. Developed the hero section.
4. Added service cards.
5. Created about and contact sections.
6. Styled the webpage using CSS.
7. Implemented responsive design using media queries.
8. Tested the webpage on different screen sizes.
9. Uploaded the project to GitHub.

### Outcome

Successfully developed a modern and responsive landing page demonstrating HTML and CSS skills. The project provides a professional user interface and adapts well to different screen sizes.

### Learning Outcomes

* HTML page structure
* CSS styling and layouts
* Flexbox concepts
* Responsive web design
* GitHub project management
